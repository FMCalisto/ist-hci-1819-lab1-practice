# Practice for Lab 1

<img src="http://web.ist.utl.pt/ist162490/pics/tecnico_logo.png" width="100#" height="auto" />

The hereby repository represents the practicing source code for the [Lab 1](https://fenix.tecnico.ulisboa.pt/disciplinas/IPM20111326/2018-2019/2-semestre/aula-1) of the [Human-Computer Interaction (HCI)](https://fenix.tecnico.ulisboa.pt/cursos/leic-a/disciplina-curricular/1529008373646) course from the 2018/2019 studies at [Instituto Superior TÃ©cnico (IST)](https://tecnico.ulisboa.pt) of [Universidade de Lisboa (ULisboa)](https://www.ulisboa.pt/) - [Portugal](https://www.portugal.gov.pt). The goal of this repository is to provide students and lecturers a source of several web development technologies. The technologies are [HTML](https://www.w3schools.com/html), [CSS](https://www.w3schools.com/css) and [JavaScript](https://www.w3schools.com/js) that will promote achieving of the presented repository goal. The general template can be found on the `src/` directory.
